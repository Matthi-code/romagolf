# Beslissing Loggen

## Formaat

Voeg toe aan `docs/memory/DECISIONS.md`:

```markdown
## [DATUM] — [Korte titel]

**Vraag:** 
Wat was de keuze die gemaakt moest worden?

**Besluit:**
Wat is er gekozen?

**Waarom:**
- Reden 1
- Reden 2

**Afgewezen alternatieven:**
- Alternatief A — waarom niet
- Alternatief B — waarom niet

**Impact:**
Welke bestanden/systemen worden beïnvloed?
```

## Wanneer Loggen

- Database keuzes
- Framework/library keuzes
- API design beslissingen
- Architectuur patronen
- Pricing/business model
- Belangrijke trade-offs

## Let Op

- Log beslissingen DIRECT, niet achteraf
- Wees specifiek over het "waarom"
- Documenteer ook wat NIET gekozen is
