# Learning Loggen

## Formaat

Voeg toe aan `docs/memory/LEARNINGS.md`:

```markdown
## [DATUM] — [Korte titel]

**Situatie:**
Wat was het probleem of de ontdekking?

**Oplossing:**
Wat werkt?

**Gotcha:**
Waar moet je op letten?

**Code voorbeeld:** (indien relevant)
\`\`\`
[code]
\`\`\`

**Bron:** [documentatie link of "eigen ervaring"]
```

## Wanneer Loggen

- API werkt anders dan verwacht
- Performance optimalisatie ontdekt
- Bug gevonden en opgelost
- Workaround nodig voor library
- Configuratie die niet obvious is
- "Dit moet ik onthouden"

## Let Op

- Log DIRECT wanneer je iets ontdekt
- Wees specifiek — vage learnings helpen niet
- Voeg code voorbeelden toe waar mogelijk
