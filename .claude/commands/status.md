# Status Check

Lees `docs/memory/PROGRESS.md` en geef een beknopte samenvatting:

```
## 📊 Project Status

**Fase:** [huidige fase]
**Focus:** [waar we mee bezig zijn]
**Blokkade:** [indien aanwezig, anders "Geen"]

**Laatste sessie:** [datum] — [wat gedaan]
**Volgende stap:** [concrete actie]

**Voltooide items:** [aantal] ✅
**Open items:** [aantal] ☐
```

Toon NIET de volledige history, alleen de huidige staat.
