# Laatste Vraag Weergeven

**Taak:** Toon de laatste vraag of opdracht die de gebruiker in deze sessie stelde.

## Output Formaat

```
📝 **Laatste vraag:**
> [de volledige vraag/opdracht van de gebruiker]

📁 **Project:** [huidige project naam]
```

## Instructies

1. Kijk terug in de conversatie naar het laatste bericht van de gebruiker
2. Toon dit letterlijk (niet samengevat)
3. Als er meerdere vragen waren, toon de meest recente

## Wanneer Nuttig

- Na lange output waarbij je vergeten bent wat je vroeg
- Bij wisselen tussen projecten/sessies
- Om te bevestigen dat Claude de juiste taak uitvoert
